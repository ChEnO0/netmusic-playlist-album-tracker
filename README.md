# 网易云歌单追踪器

一个用于追踪网易云音乐歌单更新情况，提取专辑信息并管理阅读状态的桌面应用程序。

## ✨ 核心功能

### 📋 歌单管理
- 获取歌单基本信息（名称、描述、歌曲数、更新时间等）
- 提取歌单中所有专辑信息（自动去重）
- 检查歌单更新（基于更新时间戳）
- 历史歌单记录存储（JSON格式）

### 🔖 RSS式阅读体验
- **已读/未读标记**：新添加的专辑自动标记为"未读"
- **点击标记**：点击专辑行即可标记为"已读"
- **批量标记**：一键标记当前歌单所有专辑为已读
- **视觉区分**：未读专辑显示为黑色加粗，已读专辑显示为灰色

### 📁 数据管理
- **离线查看**：点击左侧历史歌单即可加载缓存专辑数据
- **自动保存**：所有操作状态自动保存，下次启动时恢复
- **数据导出**：将专辑列表导出为CSV文件
- **链接操作**：复制专辑链接或直接打开浏览器

### 🖥️ 智能界面
- **左侧历史栏**：显示所有查询过的歌单（名称、专辑数）
- **右键删除**：右键点击歌单行删除历史记录
- **智能填充**：点击左侧歌单自动填入右侧ID框
- **链接提取**：从网易云链接自动提取歌单ID

## 🚀 快速开始

### 方法一：使用安装脚本（推荐）

**macOS/Linux**：
```bash
# 给脚本添加执行权限（仅首次运行需要）
chmod +x setup.sh run.sh

# 运行安装脚本
./setup.sh

# 运行程序
./run.sh
```

**Windows**：
1. 双击 `setup.bat` 运行安装程序
2. 安装完成后，双击 `run.bat` 运行程序

### 方法二：手动安装

#### 前提条件
- Python 3.8+
- pip

#### 安装步骤
1. 克隆或下载本项目
2. 创建虚拟环境（必需）：
   ```bash
   python3 -m venv venv
   ```
3. 激活虚拟环境：
   ```bash
   # macOS/Linux
   source venv/bin/activate
   
   # Windows
   venv\Scripts\activate
   ```
4. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

### 界面详解

#### 🎵 右侧 - 歌单操作区
1. **歌单链接输入**：粘贴网易云歌单链接，点击"提取ID"自动识别
2. **歌单ID输入**：手动输入或自动填充的歌单ID（如：3778678）
3. **历史记录下拉**：快速选择之前查询过的歌单
4. **功能按钮**：
   - **获取歌单信息**：获取最新专辑列表
   - **检查更新**：对比上次查询，检测新专辑
   - **清除**：清空显示内容

#### 📊 右侧 - 专辑显示区
1. **歌单信息面板**：显示歌单详细信息和ID
2. **专辑表格**：显示所有专辑信息（名称、艺术家、链接）
3. **操作按钮**：
   - **复制选中链接**：复制选中的专辑链接
   - **打开选中链接**：在浏览器中打开专辑页面
   - **导出专辑列表**：将专辑列表保存为CSV文件
   - **标记为已读**：将当前歌单所有专辑标记为已读

#### 📚 左侧 - 历史歌单栏
1. **歌单列表**：显示所有查询过的歌单（按固定状态排序）
   - **歌单名称**：点击自动填入右侧ID框并加载缓存数据
   - **专辑数**：歌单中包含的专辑数量
2. **右键菜单**：右键点击歌单行→"删除歌单"删除历史记录
3. **控制按钮**：
   - **刷新历史**：重新加载历史歌单列表
   - **清除所有历史**：清空所有历史记录（不可恢复）

### 📝 操作流程

#### 首次使用
1. 运行程序，界面启动
2. 在右侧输入歌单ID或链接
3. 点击"获取歌单信息"
4. 程序获取专辑列表并显示

#### 日常使用
1. 启动程序，左侧显示历史歌单
2. **点击左侧歌单**：快速加载缓存的专辑数据
3. **点击"检查更新"**：检测是否有新专辑
4. **点击未读专辑**：标记为已读（颜色变灰）
5. **右键歌单删除**：管理历史记录

#### 专辑管理
- **标记已读**：点击专辑行或使用"标记为已读"按钮
- **链接操作**：选择专辑后复制或打开链接
- **数据导出**：将专辑列表导出为CSV文件

### 🎯 使用技巧
1. **快速切换歌单**：直接点击左侧历史歌单，无需重复输入ID
2. **离线查看**：即使没有网络，也可以查看缓存的歌单数据
3. **批量操作**：使用"标记为已读"一次性处理所有未读专辑
4. **链接提取**：直接粘贴网易云链接，自动提取ID

### 使用的库
- `pyncm` - 网易云音乐Python API
- `PyQt5` - 图形界面框架
- `requests` - HTTP请求库

### 数据存储
应用程序将所有数据保存在 `data/playlist_history.json` 文件中：
- 歌单基本信息（名称、描述、歌曲数等）
- 专辑快照（所有专辑ID）
- 专辑详细信息（名称、艺术家、链接）
- 未读专辑列表
- 历史记录（最近10次查询）

## ⚠️ 故障排除

### 常见错误1：ModuleNotFoundError: No module named 'PyQt5'
**原因**：未激活虚拟环境或依赖未安装。

**解决方案**：
1. 确保已激活虚拟环境：
   ```bash
   source venv/bin/activate  # macOS/Linux
   venv\Scripts\activate    # Windows
   ```
2. 重新安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 使用启动脚本（自动处理环境）：
   ```bash
   ./run.sh  # macOS/Linux
   run.bat   # Windows
   ```

### 常见错误2：无法获取歌单信息
**可能原因**：
1. 网络连接问题
2. 歌单ID错误
3. 网易云音乐API限制

**解决方案**：
1. 检查网络连接
2. 验证歌单ID是否正确（可使用公开歌单测试，如：3778678）
3. 等待一段时间后重试

### 常见错误3：程序启动后立即退出
**可能原因**：
1. 系统缺少图形界面环境
2. PyQt5版本不兼容

**解决方案**：
1. 确保系统有桌面环境
2. 更新PyQt5：`pip install --upgrade PyQt5`

## 📋 注意事项

1. **必须激活虚拟环境**运行，否则会出现依赖错误
2. 本程序仅用于学习目的，请勿频繁请求API
3. 网易云音乐API可能随时变更，导致功能失效
4. 数据保存在 `data/playlist_history.json` 中，定期备份
5. 清除历史记录操作不可恢复，请谨慎使用

### 获取歌单ID
1. 在网易云音乐网页版打开歌单
2. 地址栏URL格式：`https://music.163.com/#/playlist?id=123456`
3. 其中的 `123456` 就是歌单ID
4. 也可使用程序内的链接提取功能
